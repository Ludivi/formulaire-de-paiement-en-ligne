# formulaire-de-paiement-en-ligne

Objectifs:

    Utiliser les évènements
    Utiliser la lisaison bidirectionnel
    Utiliser les modifiers

 

Sujet: 

Dans cet atelier, vous allez développer un formulaire de paiement en ligne par carte bancaire. Ce formulaire devra respecter les contraintes suivantes:

- les champs devront être contenus dans un formulaire

- l'utilisateur devra saisir un numéro de carte bancaire contenant 16 chiffres

- l'utilisateur devra sasir un CVC contenant 3 chiffres

- l'utilisateur devra choisir une date d'expiration valide

- en temps réel, les bordures des éléments <input> devront se colorer en vert si la contrainte de chaque champs est respectée et en rouge sinon

- le bouton de paiement permettant la validation du formulaire ne devra s'afficher que si les 3 contraintes sont respectées

- lors de la validation du formulaire, la page ne devra pas recharger, le formulaire devra disparaitre et être remplacé par un message de remerciement

- appuyer sur la touche alt+entrée sur n'importe quel <input> devra pré-remplir les champs avec des données de valides
